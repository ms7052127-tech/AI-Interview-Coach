# 🎯 AI Interview Coach

An AI-powered interview preparation platform built with React, Node.js, Express, MongoDB, and Google Gemini AI.

---

## ✨ Features

- **AI Question Generation** – Domain-specific questions generated by Google Gemini
- **5 Interview Domains** – Web Development, JavaScript, React, Node.js, HR Questions
- **3 Difficulty Levels** – Easy, Medium, Hard
- **2-Minute Timer** – Per question with auto-submit on expiry
- **Instant AI Feedback** – Score (0–10), feedback, strengths, improvements, ideal answer
- **Interview Results** – Full session breakdown with per-question analysis
- **History Tracking** – Review all past interview sessions
- **JWT Authentication** – Secure login/register with bcrypt hashing
- **Responsive UI** – Dark theme with modern design

## 🚀 Quick Start (Local Development)

### Prerequisites

- Node.js v18+
- MongoDB (local or Atlas)
- Google Gemini API key — get one free at [aistudio.google.com](https://aistudio.google.com)

---

### 1. Clone & Install

```bash
git clone https://github.com/your-username/ai-interview-coach.git
cd ai-interview-coach

# Install server dependencies
cd server
npm install

# Install client dependencies
cd ../client
npm install
```

---

### 2. Configure Environment Variables

**Server** — create `server/.env`:

```env
PORT=5000
MONGODB_URI=mongodb://localhost:27017/ai-interview-coach
JWT_SECRET=your_super_secret_key_at_least_32_chars
JWT_EXPIRES_IN=7d
GEMINI_API_KEY=your_gemini_api_key_here
CLIENT_URL=http://localhost:5173
NODE_ENV=development
```

**Client** — create `client/.env`:

```env
VITE_API_URL=http://localhost:5000/api
```

---

### 3. Run the App

**Terminal 1 — Backend:**
```bash
cd server
npm run dev
# Server starts on http://localhost:5000
```

**Terminal 2 — Frontend:**
```bash
cd client
npm run dev
# App opens at http://localhost:5173
```

## 📡 API Endpoints

| Method | Endpoint | Auth | Description |
|--------|----------|------|-------------|
| POST | `/api/auth/register` | ❌ | Register new user |
| POST | `/api/auth/login` | ❌ | Login user |
| GET | `/api/auth/me` | ✅ | Get current user |
| GET | `/api/questions/generate` | ✅ | Generate AI questions |
| POST | `/api/answers/submit` | ✅ | Submit answer + get feedback |
| GET | `/api/feedback/:id` | ✅ | Get single feedback |
| GET | `/api/feedback/interview/:id` | ✅ | Get all feedback for interview |
| GET | `/api/interviews/history` | ✅ | Get user's interview history |
| GET | `/api/interviews/:id` | ✅ | Get single interview |
| POST | `/api/interviews/:id/complete` | ✅ | Complete interview & get results |

### User

```js
{ text, domain, difficulty, interviewId, order }
```

### Answer
```js
{ userId, questionId, interviewId, answerText, isTimedOut, timeTaken }
```

### Feedback
```js
{ answerId, questionId, userId, interviewId, feedback, score, suggestions[], correctAnswer, strengths[] }
```

## ☁️ Deployment

### Frontend → Vercel

1. Push `client/` to GitHub
2. Import project on [vercel.com](https://vercel.com)
3. Set environment variable: `VITE_API_URL=https://your-render-backend.onrender.com/api`
4. Deploy

### Backend → Render

1. Push `server/` to GitHub
2. Create new **Web Service** on [render.com](https://render.com)
3. Set environment variables:
   - `MONGODB_URI` – your MongoDB Atlas connection string
   - `JWT_SECRET` – a long random secret
   - `GEMINI_API_KEY` – your Gemini API key
   - `CLIENT_URL` – your Vercel frontend URL
4. Build command: `npm install`
5. Start command: `node index.js`

### Database → MongoDB Atlas

1. Create free cluster at [mongodb.com/atlas](https://mongodb.com/atlas)
2. Create database user
3. Whitelist all IPs (`0.0.0.0/0`) for Render
4. Copy connection string to `MONGODB_URI`

## 🔑 Getting a Gemini API Key

1. Go to [aistudio.google.com/app/apikey](https://aistudio.google.com/app/apikey)
2. Sign in with Google
3. Click **Create API Key**
4. Copy and add to `server/.env` as `GEMINI_API_KEY`

The free tier includes generous quota for development.
